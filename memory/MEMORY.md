## ⛔ Windows .bat 文件写入铁律（2026-05-12 血泪教训）

**问题**：Git Bash 里反斜杠 `\` 是转义符，写 Windows 路径到 .bat 文件时 `\K`、`\H`、`\.` 会被吞掉。
**后果**：路径 `D:\Knowledge\.workbuddy` 变成 `D:Knowledge.workbuddy`，.bat 双击报错。
**犯了几次**：两次。昊鹏发火了。
**正确做法**：用 Node.js 的 `String.fromCharCode(92)` 生成反斜杠，绝不硬编码 `\`。
**禁止**：禁止用 Git Bash 的 `cat >` 或 `echo` 写包含 Windows 路径的 .bat 文件。

## MJ提示词自动收录流程（Mode A，无需确认）

触发条件：昊鹏发来MJ出图结果（图片+提示词），或发来包含MJ提示词的网址

处理步骤：
1. 图片处理：缩略图，宽不超过500px
2. 提示词：自动中英互译（昊鹏发英文→出中文；发中文→出英文；发网址→抓取后翻译）
3. 关键点提炼：
   - **艺术家和艺术风格**（关键）→ 明确列出
   - **技术/渲染器**（UE5、Octane Render、厚涂、白描、像素艺术等）→ 明确列出
4. 直接写入飞书云文档 `https://mokylin.feishu.cn/docx/V3x2dQvKGoin4nxwtJncbyUznCh`
5. 写入格式参照文档现有格式：图片 + **英文：** + **中文：** + **提示词关键点：**
6. 如发来的是网址 → 自动抓取页面中的提示词和图，按上述格式收录

工具：`lark-unified` skill → `lark-cli docs +update`
