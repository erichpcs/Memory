## bat分支名Bug修复（下午第二轮）
- **根因**：C盘`.workbuddy`仓库在`master`分支，但bat文件写死了`git push/pull origin main`
- 昊鹏多次运行`Up_CPan.bat`都失败，因为push到不存在的`main`分支
- 已手动`git push origin master`确认C盘首次推送成功
- 修复方案：4个bat全部改为动态检测分支名
  - `git branch --show-current` 获取当前分支
  - 兜底默认值`master`
  - `origin !BRANCH!` 替代 `origin main`
- 4个文件均已验证无`origin main`残留
- **教训**：永远不要硬编码分支名，必须动态检测
